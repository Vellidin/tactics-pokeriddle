# Sistema de Builds

## Filosofía

Las Builds no existen al comenzar la partida.

Se descubren.

Se construyen.

Se adaptan.

El jugador nunca debe poder planificar una Build perfecta desde el minuto uno.

Debe improvisar.

---

## Componentes

Una Build está formada por:

Pokémon.

Reliquias.

Objetos.

Movimientos.

Eventos.

Juramentos.

Estados.

Clima.

Terreno.

Todo suma.

## Builds híbridas

El jugador puede mezclar varias.

Ejemplo.

Agua + Hielo.

Fuego + Críticos.

Veneno + Fantasma.

Planta + Curación.

No existen límites.

---

## Objetivo

Ninguna Build debe ser claramente superior.

Todas deben poder completar una Run.
