# Meta Progression

## Filosofía

Cada Run comienza desde cero.

Sin embargo, la cuenta del jugador mantiene un progreso permanente.

El objetivo es que cada partida tenga valor, incluso si termina en derrota.

Nunca debe sentirse que una Run ha sido "una pérdida de tiempo".

---

# Lo que se mantiene entre Runs

• Perfil del jugador
• Estadísticas
• Logros
• Colección Pokémon
• Shinies capturados
• Entrenadores desbloqueados
• Skins
• Fondos
• Música
• Avatares
• Cosméticos
• Regiones desbloqueadas
• Reliquias descubiertas

Nunca se mantienen:

• Nivel
• Objetos
• Dinero
• Equipo
• Vida
• Inventario

---

# Nivel de Entrenador

La cuenta posee un nivel permanente.

Se obtiene experiencia por:

Completar Runs

Capturar Pokémon

Descubrir reliquias

Derrotar líderes

Completar logros

El nivel únicamente desbloquea contenido.

Nunca aumenta el poder.
