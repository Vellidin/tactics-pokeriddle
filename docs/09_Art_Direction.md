# Dirección Artística

# Objetivo

El juego debe tener identidad propia.

No parecer un fan game.

No parecer una copia.

Debe sentirse profesional.

---

# Paleta

Negros.

Grises azulados.

Azules.

Morados.

Pequeños acentos de color según el tipo Pokémon.

---

# Iconografía

Todo mediante SVG.

Nunca emojis.

Nunca iconos inconsistentes.

Todos deben compartir el mismo grosor de línea.

El mismo estilo.

---

# Fondos

Cada zona posee un fondo exclusivo.

Pueblo Paleta.

Bosque Verde.

Monte Moon.

Ciudad Celeste.

Central Rocket.

Liga.

Todos con profundidad.

Capas.

Parallax.

---

# Pokémon

Sprites oficiales.

Nunca redibujados.

No aplicar filtros agresivos.

Mantener calidad.

---

# Iluminación

Muy importante.

Todo el juego utiliza iluminación.

Sombras suaves.

Brillos.

Reflejos.

Cristales.

---

# Partículas

Las partículas ayudan al combate.

No deben distraer.

Ejemplos.

Hojas.

Chispas.

Agua.

Humo.

Arena.

Cristales.

Luz.

---

# Colores por tipo

Cada tipo tiene una identidad.

Planta.

Verdes.

Fuego.

Rojos.

Agua.

Azules.

Eléctrico.

Amarillos.

Fantasma.

Morados.

Acero.

Grises.

Siempre consistentes.

---

# Escenarios

Cada escenario modifica la iluminación.

Bosque.

Luz verde.

Volcán.

Luz naranja.

Cueva.

Oscuridad.

Laboratorio.

Luz blanca.

Mar.

Azul.

---

# Menús

Todos los paneles flotan.

Nunca ocupan toda la pantalla.

Siempre dejan ver el fondo.

---

# Tipografía

Una única fuente.

Muy legible.

Preparada para móvil.

Sin estilos extravagantes.

---

# Sensación

La palabra que define toda la dirección artística es:

Elegancia.

No saturación.

No exceso de efectos.

No interfaces infantiles.

El jugador debe sentir que está jugando un RPG moderno de alta calidad.
