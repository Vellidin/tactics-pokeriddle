# Monetización

## Filosofía

La monetización nunca debe afectar al equilibrio del juego.

Un jugador gratuito debe poder completar absolutamente todo el contenido.

Nunca existirán Pokémon exclusivos de pago.

Nunca existirán Reliquias más fuertes para usuarios Premium.

Nunca existirá Pay To Win.

El dinero únicamente desbloquea:

Cosméticos.

Comodidad.

Personalización.

Contenido opcional.

---

# Usuario Gratuito

Puede:

Jugar Runs.

Completar Kanto.

Capturar todos los Pokémon.

Conseguir todas las Reliquias.

Completar la Pokédex.

Participar en Rankings.

Realizar Daily Runs.

Todo sin pagar.

---

# Premium

Incluye.

Runs ilimitadas.

Skins exclusivas.

Tableros exclusivos.

Fondos exclusivos.

Marcos.

Animaciones.

Avatares.

Mayor espacio para guardar Seeds favoritas.

Historial completo.

Estadísticas avanzadas.

Sin anuncios.

Nunca ventajas jugables.

---

# Tienda

La tienda únicamente vende.

Skins.

Fondos.

Animaciones.

Emotes.

Avatares.

Pases de temporada.

Nunca objetos de combate.

---

# Pase de temporada

Cada temporada incluye.

Skins.

Fondos.

Música.

SVG.

Cosméticos.

Nunca poder.

---

# Anuncios

Solo usuarios Free.

Nunca durante combate.

Nunca durante un Boss.

Solo:

Tras una Run.

Desde el menú.

Opcionalmente para obtener una recompensa cosmética menor.

---

# Objetivo

El jugador nunca debe sentir que ha perdido por no pagar.

Debe pagar porque quiere apoyar el proyecto.
